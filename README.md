# Salad Info Extractor

Salad Info Extractor is a Python script that extracts and displays information from log files generated by Salad. It specifically targets the latest log files and extracts information about bandwidth usage, predicted earnings, workload details, and wallet balance.

## Disclaimer

This tool is for educational purposes only. Use it at your own risk.

## Installation

### Running the Compiled Version

1. Download the executable file.
2. Run the executable file.

### Running from Source

1. Ensure that Python is installed on your machine.
2. Clone the repository or download the source code.
3. Install the necessary Python libraries by running `pip install -r requirements.txt` in your terminal.
4. Run the script by typing `python Salad-Info.py` in your terminal.
